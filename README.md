# calculator1

Trang tính đơn ShopeeFood bằng một file HTML tĩnh.

## Cách dùng

Mở `calculator.html` bằng trình duyệt để sử dụng trực tiếp.

## Chức năng

- Nhập số đơn ghép, đơn lẻ, số km và tiền tip.
- Tính tiền đơn, phí đơn con, phí thứ tự, tiền xăng và thực nhận.
- Lưu lịch sử vào trình duyệt.
- Xuất lịch sử ra CSV.
